IUS Official Drum Sequencer

Technical hardware layout

Device overview

The device is a real time hardware sampler and sequencer built around a Daisy Seed 2 DFM main processor, an external stereo audio codec, a dedicated analog audio board, a separate controls board, a left side main monitor section, and an internal 3 channel stereo mixer with stereo master output.

The system is divided into four hardware sections.

Power rules

Digital power must be separated from analog power
Codec analog rail must be clean
Headphone and output stage must use clean analog power
Analog input section must be physically separated from noisy control scanning and digital switching areas
Grounding must be planned so analog return paths stay quiet
